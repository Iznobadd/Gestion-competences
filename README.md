## Run Locally

Clone the project

```bash
  git clone https://github.com/Iznobadd/Gestion-competences.git
```

Go to the project directory

```bash
  cd Gestion-competences
```

Install dependencies
```bash
  composer install
```
```bash
  npm install
```

Start the server

```bash
  symfony serve
```
```bash
  yarn encore dev-server
```
